**aico 是你构建 AI 团队的方式。**

传统的 AI 助手在你需要专业工作流程时会遇到困难。你最终会反复编写自定义提示词、重复指令，并在会话之间丢失上下文。

这正是 aico 要解决的问题。它基于以下原则构建：

- **员工，而非提示词：** 具有预置技能、命令和领域专业知识的 AI 协作者。
- **技能优于指令：** 可复用的能力，根据上下文自动触发。
- **开源：** 一切透明且可定制。
- **平台无关：** 支持 Claude Code、Codex 和其他 AI 编码助手。

## 员工

aico 引入了"AI 员工"的概念 - 包含技能、命令和文档的包，用于扩展 AI 编码助手。

每个员工都有特定的角色：

- **PM（产品经理）：** 产品需求、用户故事、头脑风暴
- **前端工程师：** UI/UX 实现、组件设计、样式
- **后端工程师：** API 开发、数据库设计、系统架构

## 技能

技能是员工的核心能力。它们根据上下文自动触发，也可以手动调用。

_当你说"我有一个产品想法..."时，PM 的头脑风暴技能会自动激活，引导你进行结构化的创意构思。_

## 命令

命令是常见工作流程的快捷方式。它们通过斜杠前缀调用，如 `/pm.plan` 或 `/frontend.design`。

_命令让你用一个动作触发复杂的工作流程。_

## 快速开始

```bash
# 全局安装 CLI（推荐）
npm install -g @the-aico/cli

# 初始化项目
aico init

# 添加 AI 员工（简写）
aico add pm

# 或使用完整名称
aico add @the-aico/pm
```

就是这样。你的 AI 助手现在拥有 PM 能力，包括头脑风暴、需求分析和用户故事编写。
